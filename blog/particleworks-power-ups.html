Particleworks Power-Ups: Python Automation for Faster Simulation Workflows

EnginSoft UK July 2025 4 min read

The Setup Tax Nobody Budgets For

Every simulation engineer knows the feeling. The physics is interesting, the geometry is ready, and then you spend two hours clicking through dialog boxes to set up gear trains, assign material properties, and configure motion constraints. It is not difficult work. It is tedious work. And tedious work is where mistakes hide.

EnginSoft UK built a set of Python automation tools for Particleworks called Power-Ups — bite-sized scripts that hook into the Particleworks Python API to eliminate the most common repetitive setup tasks. Each one solves a specific problem, runs in seconds, and does exactly what it says on the tin.

01

Spot Polygon Motion Constraint Errors

When you import complex assemblies, it is easy for a component's motion constraint to reference the wrong coordinate system or rotation axis. This Power-Up visualises min/max displacement markers on every moving polygon, instantly revealing CAD import mismatches that would otherwise surface only after hours of wasted compute time.

02

Automate Gearbox Setup

A typical gearbox model involves dozens of parent-child relationships, gear ratios, and rotation directions. Configuring them manually is slow and error-prone. This script reads the gear train definition and applies all parenting hierarchies and motion rules in seconds, turning a half-day setup task into a single click.

03

Build a Fluid Property Library

Every new project starts with the same ritual: dig through datasheets, copy viscosity curves, enter density and surface tension values. This Power-Up lets you build and maintain a centralised fluid property library. Click once, pick your fluid, and the correct temperature-dependent properties are imported directly into the simulation. No more transcription errors, no more hunting for that PDF.

04

Calculate Gear Power Loss

Churning and windage losses in gearboxes depend on rotational speed, oil level, and gear geometry. This Power-Up handles variable RPM profiles across multiple gear meshes, extracting torque-averaged power loss data directly from the simulation results without the spreadsheet gymnastics.

05

Circumferential Velocity in the GUI

Checking circumferential velocity used to mean exporting particle data to Excel, computing radial positions, and back-calculating tangential components. This Power-Up adds the quantity directly to the Particleworks GUI as a derived field, so you can visualise and interrogate it without leaving the application.

All Power-Ups are available through EnginSoft UK and can be customised to fit your specific workflow requirements.
